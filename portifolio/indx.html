Módulo 1 - Conceitos básicos

Neste módulo, assim como o título ja diz, aprendi alguns conceitos básicos, como utilizar um software para editar códigos e algumas tags.

  • O que é HTML e para que ele serve?

    • De forma resumida, HTML é uma linguagem de marcação, que funciona para estruturar o conteúdo de um site. Em sua edição mais atual, o HTML5, devemos nos preocupar com a semântica e fazer uso consciente das tags para dar sentido ao conteúdo do site.

  • Qual software escolhi para estudar?

    • O software utilizado para editar os meus códigos HTML será o Visual Studio Code. Nele fiz a instalção de algumas extensões para me ajudar na edição dos códigos. Segue os nomes das extensões utilizadas até o momento. "Code Runner", "ESLint", "Live Server", "OM Theme (A Darker Dracula Theme)"

  • Algumas Tags de HTML

    • As tags são um conjunto de palavras entre sinais de colchete angular, as tags podem ter atributos e valores, uma mesma tag pode ter vários parâmetros, cada um com seu valor. Algumas tags não possuem a necessidade de fechamento. As primeiras tags apresentadas no início do curso foram: <h1>, <p>, <img> e <hr>

  • Atributos

      • Os atributos são valores adicionados nas tags. Os primeiros atributos que aprendi são esses "href" e "src" ambos podem ser utilizados dentro da tag <img href="link de alguma coisa", src="caminho de alguma coisa">
